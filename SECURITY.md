# Політика безпеки (Security Policy)

Цей проєкт на MicroPython розробляється одноосібно.

## Як повідомити про вразливість
Якщо ви знайшли проблему з безпекою, будь ласка, **не створюйте публічне Issue**. 
Натомість використовуйте вбудовану функцію GitHub для повідомлення про вразливості:
Перейдіть на вкладку "Security" цього репозиторію та натисніть "Report a vulnerability". 
Це дозволить нам обговорити проблему приватно.
